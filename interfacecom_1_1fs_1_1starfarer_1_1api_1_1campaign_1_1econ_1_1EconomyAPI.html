void com.fs.starfarer.api.campaign.econ.EconomyAPI.doubleStep ( )

Equivalent to calling nextStep() two times.

Should only be called from UI interactions, not from scripts that run in the background. Calling this method is likely to take longer than the time to render a single frame.

void com.fs.starfarer.api.campaign.econ.EconomyAPI.tripleStep ( )

Equivalent to calling nextStep() three times. Generally enough or almost enough to fully stabilize the economy.
